Quiero diseñar un proyecto en Cisco Packet Tracer para una materia de Redes 2. El objetivo es simular la optimización de latencia en una red, inspirado en herramientas como WTFast (sin necesidad de replicarlo completamente).

Necesito que me ayudes a:

1. Diseñar una topología de red que incluya:

   * Al menos 3 routers
   * 2 rutas posibles entre un cliente y un servidor
   * Diferentes valores de delay y ancho de banda en cada ruta

2. Configurar:

   * Direccionamiento IP completo
   * Un protocolo de enrutamiento dinámico (OSPF o EIGRP)
   * Manipulación de métricas para influir en la selección de rutas

3. Simular latencia:

   * Agregar delay manual en interfaces
   * Explicar cómo esto afecta el tráfico

4. Implementar QoS:

   * Clasificar tráfico (simular tráfico de “juego”)
   * Priorizar ese tráfico usando class-map y policy-map
   * Aplicar políticas en interfaces

5. Validación:

   * Comandos para probar conectividad (ping, traceroute)
   * Comparación antes y después de aplicar QoS y optimización de rutas

6. Explicación teórica:

   * Relacionar el proyecto con el concepto de reducción de ping
   * Explicar cómo esto simula herramientas de optimización de red

Quiero que incluyas:

* Diagrama lógico de la red (puede ser en texto)
* Configuración paso a paso en routers (comandos Cisco)
* Explicaciones claras para cada parte
* Buenas prácticas

El nivel debe ser intermedio (estudiante universitario de redes).
